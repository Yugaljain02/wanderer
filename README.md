# Wanderer - Airbnb Style Travel Listing App

Wanderer is a travel listing web application similar to Airbnb where users can create, view, edit, and delete property listings.

## Features

- User authentication
- Add listing
- Edit listing
- Delete listing
- Image upload
- MVC structure

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Cloudinary

## Run project

npm install
npm start

Open browser:
http://localhost:3000
